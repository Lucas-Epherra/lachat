# Estudio Jurídico Lachat — Landing Page

Landing page profesional para **Estudio Jurídico Lachat**, diseñada para presentar servicios legales de manera clara, elegante y orientada a la conversión.

El sitio está enfocado en asesoramiento legal para contratos de locación, arrendamientos rurales, contratos comerciales, revisión contractual e intimaciones por carta documento.

## Vista general

Este proyecto busca construir una presencia digital sólida, moderna y confiable para un estudio jurídico boutique.

La identidad visual sigue una estética editorial legal premium: fondo marfil, azul petróleo profundo, dorado champagne sutil, curvas orgánicas, detalles botánicos finos, tipografía serif protagonista y una composición limpia con mucho aire visual.

## Servicios destacados

- Contratos de locación urbana
- Contratos comerciales
- Arrendamientos rurales
- Revisión preventiva de contratos
- Intimaciones por carta documento
- Asesoramiento legal online y presencial

Instalación y uso local

1. Clonar el repositorio:

```git clone https://github.com/TU_USUARIO/lachat.git```

2. Entrar al proyecto:

```cd lachat```

3. Instalar dependencias:

```npm install```

4. Ejecutar en entorno local:

```npm run dev```

5. Abrir en el navegador:

```http://localhost:5173```

6. Scripts disponibles

```npm run dev```

7. Ejecuta el proyecto en modo desarrollo.

```npm run build```

8. Genera la versión optimizada para producción.

```npm run preview```

Permite previsualizar localmente la build de producción.
