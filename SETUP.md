# SETUP.md — Estudio Jurídico Lachat

## Identidad visual

Estética editorial / boutique legal premium.

La landing debe transmitir:
- Elegancia
- Claridad
- Cercanía profesional
- Firmeza jurídica
- Femenino con fuerza
- Confianza y prevención

## Paleta

- Fondo principal: `#FFF9EF`
- Azul profundo / petróleo: `#082E3A`
- Dorado champagne: `#D7B56D`
- Texto secundario: `#355966`
- Crema decorativo: `#EFE4D2`
- Crema suave: `#F1E7D8`
- Blanco cálido: `#FFFDF7`

## Reglas de diseño

- Usar azul profundo solo para botones, bloques de contraste y secciones importantes.
- No usar formas azul oscuro detrás de títulos o contenido principal.
- Los ornamentos grandes deben ser crema, marfil o dorado suave.
- El dorado debe funcionar como acento fino, no como protagonista.
- Mantener mucho aire visual.
- Usar tipografía serif para títulos grandes.
- Usar texto sans o simple para contenido secundario.
- Incorporar curvas orgánicas claras.
- Usar botánica fina y sutil.
- Evitar estética corporativa fría o genérica.

## Componentes visuales

Elementos permitidos:
- Curvas orgánicas
- Líneas doradas finas
- Divisores con rombo central
- Sellos/labels editoriales
- Cards suaves con borde dorado tenue
- Bloques azul petróleo para contraste
- Monograma `L`
- Detalles botánicos delicados

Elementos a evitar:
- Íconos legales genéricos excesivos
- Balanza gigante
- Martillo de juez como protagonista
- Fondos grises corporativos
- Dorado saturado
- Sombras pesadas
- Ornamentos que compitan con el texto

## Tono verbal

Claro, directo y profesional.

Debe sentirse:
- Preventivo
- Confiable
- Cercano
- Firme
- Ordenado

Ejemplo de tono:
> Contratos claros. Reclamos firmes.

> Antes de firmar, asesorate. Antes de reclamar, intimá correctamente.

> Un contrato bien redactado puede evitar un conflicto mañana.
